# Hello Github Skeleton! 💀
해당 프로젝트는 Github를 이용한 팀 프로젝트를 위한 스켈레톤 프로젝트입니다.

## What is in it?
- 다양한 Github 템플릿
    - 이슈 템플릿
    - 풀 리퀘스트 템플릿
- 유용한 Github Actions
    - Pull Request Assignee, Reviewer 자동 할당
    - Pull Request Label 자동 할당
- Github Convention
    - 브랜치 네이밍 규칙
    - 커밋 메시지 규칙
    - PR 메시지 규칙

# How to use?
해당 프로젝트를 Fork하거나, Clone 하거나, Copy & Paste하여 사용하면 됩니다.
